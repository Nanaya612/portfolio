## Essa Aplicação foi feita para sumarizar alguns dos meus projetos e algumas das minhas habilidades desenvolvidas 
# Projetos Apresentados: 

- ## To do List 

    Uma aplicação para montar e organizar um lista de tarefas.

- ## Blog Pescaria

    Uma aplicação que simula uma página de blog, feita para eu experimentar algumas funções do Next.js.

- ## Previsão Temporal 
    
    Uma aplicação que informa a previsão temporal diária de uma cidade específica.

# Como Rodar: 

## Localmente
- Clone o repositório com o comando `git clone https://github.com/Nanaya612/portfolio`
- Instale as dependências com o comando `npm install`
- Faça o build com o comando `npm run build`
- Inicie com o comando `npm run start`
